Le document "proposal-rt-sigma.pdf" présente une proposition de recherche pour le projet **Real-time Signal Interference GNSS-network Monitoring and Analysis `rt-sigma`**. Ce document se base sur une proposition conjointe `"sigma"`de l'ERM et l'IGN et ce projet  devrait commencer cette année. 

La différence entre ces projets est l'aspect temps réel du projet. `rt-sigma` se concentre sur la surveillance et l'analyse en temps réel des interférences de signal dans les réseaux GNSS, tandis que le projet `sigma` se base sur l'analyse post-traitement des données RINEX.

Ce document `rt-sigma` pourrait être utilisé de 2 façons différentes :
- il sert comme base d'une proposition de recherche que M3SB pourrait soumettre à des organismes de financement pour obtenir des fonds pour le projet.
- certaines sections peuvent être utilisées dans le cadre de l'accord que M3SB a avec WALCORS pour détecter des anomalies dans la station WAVRE.

Je suis prêt à t'en parler et définir ensemble les prochaines étapes.
